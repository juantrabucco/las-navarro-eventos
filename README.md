# Las Navarro Eventos — Landing Page

Sitio web oficial de Las Navarro Eventos, organizadoras de casamientos
y celebraciones en Buenos Aires, Argentina.

## Stack
- HTML5 semántico
- CSS3 (custom properties, grid, flexbox)
- JavaScript vanilla (IntersectionObserver, scroll events)
- Google Fonts: Cormorant + DM Sans

## Assets
Los assets de marca están en `/assets/logos/` e `/assets/icons/`.
Las fotos de eventos van en `/assets/images/` organizadas por sección.

## Deploy
Compatible con Vercel, Netlify y GitHub Pages sin configuración adicional.
Abrir `index.html` directamente en el browser para desarrollo local.
